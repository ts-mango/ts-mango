## 介绍

这是一个可爱的芒果。

TsMango 是一个用 typescript 编写的实用函数库.

## 为什么 TsMango?

- 🔥 使用 TypeScript
- 🔔 强类型
- ✅ 按需引入
- ✔️ 严格校验
- 💪 健壮的代码
- 🚩 实用函数
- 🚀 提高开发效率

## 下载

Pnpm

```shell
pnpm add ts-mango
```

Npm

```shell
npm i ts-mango
```

Yarn

```shell
yarn add ts-mango
```

## 实用

```ts
// Complete introduction
import TsMango from 'ts-mango'
TsMango.isNumber()

// Import in as needed
import { isNumber } from 'ts-mango'
isNumber()
```
